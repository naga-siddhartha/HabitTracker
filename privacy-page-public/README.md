# Habits app — Privacy Policy (public)

This folder is a copy of the privacy policy content. The live policy is hosted in the **Habit-Tracker-App** repo and served via GitHub Pages.

**Live URL:** https://naga-siddhartha.github.io/Habit-Tracker-App/

That URL is set in the app’s **Info.plist** (`PrivacyPolicyURL`) and should be used in **App Store Connect** → App Information → Privacy Policy URL.

To update the live policy: push changes to the [Habit-Tracker-App](https://github.com/naga-siddhartha/Habit-Tracker-App) repo (or use the `Habit-Tracker-App-push/` folder in this project to commit and push).
