# Privacy policy – internal copy

**Canonical v2 policy:** `docs/privacy.html` in this repo.

This file is the source of truth for Ritual Log v2’s privacy policy. It includes:

- Sign in with Apple (user identifier, optional name/email)
- Account data storage (Keychain on device)
- Optional future iCloud/CloudKit sync and transmission (Section 5)
- Account deletion and how to revoke Apple ID access (Section 3)
- Last updated date (e.g. March 2026)

**When you are ready to publish:** Copy `privacy.html` to the repository or host where your public-facing policy lives (e.g. `Habit-Tracker-App-push/index.html` or `privacy-page-public/index.html`), then deploy. Ensure the app’s **Privacy Policy URL** (Info.plist key `PrivacyPolicyURL` and App Store Connect) points to that deployed page.

Do not edit the public copy without updating this internal copy first, so the repo stays in sync.
